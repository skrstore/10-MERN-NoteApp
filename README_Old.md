# MERN Todo App

## Technologies and Tools

- TextEditor - Visual Studio Code
- VSCode Extensions
  - Prettier - for code formatting
- Development OS - Window 10
- Frontend - React
  - Package manager - Yarn
  - Style - Scss
- Backend - NodeJS(ExpressJS)

## Usage

```cmd
git clone https://github.com/skr571999/MERNTodoApp.git

cd client
yarn install
npm start

cd server
npm install
npm start
```
